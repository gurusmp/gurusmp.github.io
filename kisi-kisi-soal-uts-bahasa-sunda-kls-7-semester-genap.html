18. mohon bantuannya kk ini soal kisi kisi uts pagi ini kk plis


No. 7
-------
Diket : 
pers. Posisi, r(t) = (5)i - (3t³ + 2t²)j

Tanya :
Perpindahan dari t = 2 sec sampai t = 5 sec, |Δr| = __?

Jawab :
Step 1
--------
cari posisi saat t = 2 sec, 
r(2) = (5)i - (3.2³ + 2.2²)j
r(2) = (5)i - (24 + 8)j
r(2) = (5)i - (32)j

Step 2
--------
cari posisi saat t = 5 sec, 
r(5) = (5)i - (3.5³ + 2.5²)j
r(5) = (5)i - (375 + 50)j
r(5) = (5)i - (425)j

Step 3
--------
vektor perpindahan, Δr = __?
Δr = r(5) - r(2)
Δr = [(5)i - (425)j] - [(5)i - (32)j]
Δr = (5)i - (425)j - (5)i + (32)j
Δr = (5)i - (5)i - (425)j + (32)j
Δr = - (393)j 

Dan besar perpindahannya adalah 
|Δr| = √[0² + (- 393)²]
|Δr| = 393 meter


No. 8
--------
Diket :
Posisi awal, A (2,5) meter →ra = (2)i + (5)j
posisi akhir, B (40,80) meter →rb = (40)i + (80)j
waktu, t = 10 menit = 10 x 60 sec = 600 sec

Tanya :
Kecepatan rata-rata, V = __?

Jawab : 
Step 1
--------
perpindahan, Δr = __?
Δr = rb - ra
Δr = [(40)i + (80)j] - [(2)i + (5)j]
Δr = (40)i + (80)j - (2)i - (5)j
Δr = (40)i - (2)i + (80)j - (5)j
Δr = (38)i + (75)j

Besar perpindahan
|Δr| = √(38² + 75²)
|Δr| = √(1444 + 5625)
|Δr| = √7069
|Δr| = 84,077 meter

Step 2
---------
Kecepatan,
V = Perpindahan/waktu
V = |Δr| / Δt
V = 84,077 / 600
V = 0,1401 m/s
