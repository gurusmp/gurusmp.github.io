Jawaban:

Berdasar Pengertiannya :

- Sistem Operasi adalah sebagai dasar agar dapat menjalankan aplikasi-aplikasi atau seperangkat program yang mengelola sumber daya perangkat keras komputer dan menyediakan layanan umum untuk aplikasi perangkat lunak.

- Sistem Operasi Jaringan adalah suatu sistem operasi yang mempunyai fungsi-fungsi jaringan.

Berdasar Fungsinya :

- Sistem Operasi berfungsi sebagai penghubung antara perangkat keras dengan aplikasi-aplikasi atau perangkat lunak.

- Sistem Operasi Jaringan berfungsi sebagai penghubung sejumlah komputer dan perangkat lainnya ke sebuah jaringan.

Berdasarkan Jenis-jenisnya :

- Jenis-jenis Sistem Operasi : SO Real Time, SO Multiuser, SO Multitasking, SO Terdistribusi.

- Jenis-jenis Sistem Operasi Jaringan : SOJ Peer to Peer dsn SOJ Client-Server.

Berdasarkan Pengoperasiannya :

- Sistem Operasi dapat beroperasi atau bekerja pada satu perangkat komputer saja atau PC (stand alone).

- Sistem Operasi Jaringan dapat bekerja atau beroperasi minimal melibatkan 2 perangkat komputer

10. macam-macam aplikasi system operasi beserta contohnya!


Pengertian Operating System (Sistem Operasi)
Secara mudah saja kita sebagai pengguna menggunakan komputer melalui media aplikasi atau yang lebih kerap disebut dengan Software (perangkat lunak). Aplikasi-aplikasi dalam komputer ini bekerja menurut OS (Operating System) yang telah ditanamkan dalam komputer, jadi tanpa adanya OS maka komputer tidak dapat digunakan. Secara gambaran aplikasi bekerja menurut OS, dan OS bekerja menurut perangkat keras komputer. Kesimpulannya sistem operasi merupakan sebuah program yang mengatur dan mengelola perangkat-perangkat keras (hardware) dari komputer dan memberikan pelayanan (sumber daya) terhadap aplikasi-aplikasi perangkat lunak. Sekarang anda bisa menyimpulkan bahwa Monitor, keyboard, speaker dan semua perangkat keras yang terdapat dalam komputer sepenuhnya dikendalikan oleh OS. OS juga memberikan pelayanan terhadap aplikasi-aplikasi yang ingin menggunakan Hardware dari komputer namun terbatas berdasarkan OS yang digunakan, misalnya aplikasi pemutar musik yang mengendalikan speaker.

Apa saja macam sistem operasi untuk komputer yang sudah ada saat ini?
Sistem operasi komputer saat ini sudah ada banyak, tak hanya komputer bahkan perangkat mobile seperti HP (Smartphone) kini juga telah menggunakan sistem operasi. Macam sistem operasi untuk HP yang populer contohnya: Android, Windows Phone, Symbian, dll. Macam-macam sistem operasi untuk komputer;

1. Windows
Microsoft Windows atau disebut dengan Windows saja merupakan salah satu sistem operasi yang ada saat ini. Windows menjadi sistem operasi yang sangat populer, bahkan bisa dikatakan hampir semua komputer pernah menggunakan OS ini. Dominasi Windows terhadap OS komputer menjadikannya semakin maju dan diperbaharui, bahkan untuk perangkat seluler telah menciptakan OS windows Phone. Kini Sistem Operasi mutakhir dari windows diberi nama Windows 8 setelah sebelumnya sukses dengan Windows 7. Beberapa versi dari windows diantaranya Windows 98, Windows 2000, Windows XP, Windows Vista, Windows 7, dan Windows 8.

2. Linux
Sistem Operasi Linux merupakan salah satu sistem operasi yang sukses. Meskipun tidak sesukes Microsoft Windows namun Berbagai perusahaan besar seperti Intel telah mengenal sistem operasi ini. Sistem operasi linux biasanya digunakan untuk server, Video game (Playstation 2, 3, 4), Ponsel, dan juga untuk komputer. Keunggulan sistem Operasi Linux dibanding windows yang paling mencolok adalah dari tingkat keamanannya. Secara gamblang akan banyak sekali orang yang bersemangat untuk mempelajari sistem operasi Windows karena hampir semua komputer menggunakan windows, namun akan sedikit yang mempelajari sistem operasi linux karena pengguna linux tidak sebanyak windows. Mungkin hal inilah yang menyebabkan sistem operasi linux dikatakan lebih aman dibanding sistem operasi windows, ibaratnya banyak yang pakai pasti banyak yang ingin mencuri sedangkan sedikit yang pakai sedikit pula yang ingin mencuri.

3. Mac OS
Anda mungkin sudah sering mendengar dengan yang namanya Windows dan Linux, namun bagaimana dengan Mac OS? Mac OS merupakan singkatan dari Macintosh Operating System, sistem operasi yang dibuat oleh aplle dan dibuat untuk komputer keluaran Macintosh. Mac OS tidak dapat di gunakan oleh PC yang memiliki basis IBM (International Business Machines Corporation).

4. Android
Sudah kenal dengan Android? saya yakin anda sudah pernah mendengar nama android, sistem operasi besutan Google ini kini telah merajai OS dari Ponsel. Sifatnya yang Open Source menjadikan OS Android bisa digunakan oleh perusahaan Ponsel dengan merk apa saja, seperti: Samsung, Sony, LG, dan masih banyak lagi. Kini sistem operasi yang telah merajai pasar Smartphone ini telah mendukung untuk dijadikan sebagai sistem operasi Komputer. Android Jelly Bean 4.3 kini telah bisa dijadikan OS komputer, meskipun demikian Google masih terus melakukan pengujian terhadap sistem operasi ini.

5. Chrome OS
Google Chrome OS, satu lagi produk yang dihasilkan oleh raksasa Internet Google. Chrome OS merupakan sistem operasi yang dapat berjalan pada komputer (perangkat keras) tertentu. Chrome OS dirancang secara khusus bagi mereka yang menghabiskan waktunya di internet. Goole Chrome OS menggunakan standar web HTML 5. Kini Browser Google Chrome juga telah merebut sebagian besar pasar dari Mozila firefox, nampaknya Google benar-benar akan menguasai di semua bidang dalam hal Teknologi Informasi.

12. apa perbedaan system' operasi jaringan berbasis text (open source) deang berbasis GUL​


Sistem operasi jaringan berbasis teks (open source) dan berbasis GUI (Graphical User Interface) memiliki beberapa perbedaan utama:

1. Antarmuka Pengguna:

- Berbasis Teks: Sistem operasi jaringan berbasis teks menggunakan antarmuka teks berbasis baris perintah, yang memerlukan pengguna untuk memasukkan perintah teks untuk berinteraksi dengan sistem. Contohnya adalah Linux dengan shell seperti Bash.

- Berbasis GUI: Sistem operasi jaringan berbasis GUI menyediakan antarmuka berbasis grafis yang lebih intuitif dengan elemen seperti ikon, menu, dan jendela. Windows dan macOS adalah contoh sistem operasi berbasis GUI.

2. Kompleksitas:

- Berbasis Teks: Sistem operasi berbasis teks sering dianggap lebih fleksibel dan kuat, tetapi mungkin memerlukan tingkat pengetahuan teknis yang lebih tinggi untuk digunakan efektif.

- Berbasis GUI: Sistem operasi berbasis GUI biasanya lebih mudah digunakan oleh pengguna awam karena tampilan grafis yang lebih intuitif.

3. Konfigurasi:

- Berbasis Teks: Konfigurasi sistem pada sistem operasi berbasis teks umumnya dilakukan melalui pengeditan berkas teks dan penggunaan perintah baris perintah.

- Berbasis GUI: Sistem operasi berbasis GUI sering menyediakan alat konfigurasi grafis yang memudahkan pengguna untuk mengatur berbagai pengaturan.

4. Penggunaan:

- Berbasis Teks: Sistem operasi berbasis teks umumnya lebih umum digunakan dalam lingkungan server dan pengembangan, di mana akses jarak jauh dan otomatisasi sering diperlukan.

- Berbasis GUI: Sistem operasi berbasis GUI lebih umum digunakan dalam lingkungan desktop dan oleh pengguna yang tidak memiliki latar belakang teknis yang kuat.

14. Bagaimana cara kerja suatu Sistem Operasi Jaringan (Network Operating System) ?


Setelah komputer selesai proses booting dan me-load himem.sys untuk mendapatkan alokasi memory di atas 1 mega, maka NOS akan di-ekeskusi secara otomatis dengan memasukkannya dalam file autoxec.bat . Biasanya dalam autoexec.bat ditambahkan line run.bat yang berupa script untuk run NOS pertama kali. Run.bat akan menjalankan perintah untuk mengenali device modem TNC dengan nomor interupt tertentu. Kemudian NOS akan di-load dengan membaca file configurasi : autoexec.nos .

3.Sistem Operasi Linux

Linux diperkenalkan pertama kali oleh Linus Torvalds pada tahun 1991 silam. Sistem operasi yang satu ini memliki sifat open-source dimana pengguna dapat memodifikasi dan mendistribusikan kembali secara bebas tanpa perlu lisensi

4.Sistem Operasi Mac OS

Mac OS atau singkatan dari Machintosh operating System adalah sistem operasi eksklusif yang hanya terdapat di komputer Apple saja. MacOS yang diluncurkan pada tahun 1984 pertama kali digunakan untuk komputer bernama LISA. LISA tak lain adalah komputer hasil racikan Steve Jobs setelah ia menempuh perjalanan spiritual ke India.

5.Sistem Operasi Unix

Sama halnya seperti DOS, Unix merupakan jenis OS komputer generasi lawas yang banyak digunakan hingga menjadi cikal bakal munculnya OS masa kini. Saat ini Unix sendiri sudah mempunyai banyak versi yang bisa dipilih oleh pengguna. Terkait keunggulan dan daya tarik utama, ada beberapa aspek yang bisa Anda coba. Namun, dari segi penggunaan Unix saat ini sudah sangat jarang digunakan.

39. apa itu yang dimaksud dengan system operasi open source sebutkan contoh system operasinya​


Sistem operasi open source adalah jenis sistem operasi yang kode sumbernya tersedia untuk umum, dapat diakses, dimodifikasi, dan didistribusikan secara bebas oleh siapa pun. Ini memungkinkan pengguna dan pengembang untuk berkontribusi pada pengembangan sistem operasi tersebut. Contoh sistem operasi open source yang terkenal termasuk:

1. Linux (contoh distribusi: Ubuntu, CentOS, Debian)

2. FreeBSD

3. Android (versi inti sistem operasi Android adalah open source)

4. OpenBSD

5. GNU Hurd

Sistem operasi open source sering digunakan di berbagai perangkat, termasuk server, komputer desktop, dan perangkat mobile, karena fleksibilitas, keamanan, dan biaya yang lebih rendah yang mereka tawarkan.
