28. Mana yang benar "Class Leaders" atau " Chairman Class " untuk bahasa inggrisnya ketua kelas?​


"Class Leaders atau Chairman Class"?

Bahasa Inggris dari ketua kelas adalah =

Chairman Class or Class Leader.

Jadi, Chairman Class dan Class Leader sama-sama artinya ketua kelas.

Tetapi lebih cocok dengan

Chairman Class

karena

Chairman=>Ketua

Class=>Kelas

Sedangkan

Class=>Kelas

Leader =>Pemimpin

Jadi yang lebih tepat adalah

CHAIRMAN CLASS
