1. Topik: Membaca Nonsastra

Perekonomian di dunia terus merosot yang disebabkan resesi di Eropa yang berkepanjangan. Hal ini membawa dampak yang sangat besar bagi perajin di Indonesia karena produknya tidak dapat diekspor bahkan gagal ekspor. Untuk mempertahankan kelangsungan hidup keluarga dan karyawannya banyak perajin kita yang beralih usaha lain.

Makna tersurat paragraf di atas adalah ….

A. Perekonomian Indonesia merosot sehingga berdampak di perekonomian dunia.

B. Dampak kemerosoton perekonomian dunia, perajin Indonesia beralih usaha lain.

C. Kegagalan mengekspor produk karena perajin tidak mampu bersaing untuk menghasilkan produk unggulan.

D. Eropa menjadi penyebab Indonesia tidak bisa ekspor produk.

Jawaban: B

Pembahasan:

Makna tersurat adalah makna terhadap sesuatu hal yang disampaikan secara jelas dan dapat dipahami sebagaimana adanya baik tertulis maupun diucapkan.

Jadi, makna tersurat paragraf di atas adalah pada pilihan jawaban B.

13. Apa saja partikel yang ada dalam bahasa jepang?


Pendahuluan

Partikel atau Joshi merupakan sebuah kata bantu kalimat yang memiliki beragam fungsi. Jumlah partikel Bahasa Jepang ada sekitar 188 partikel, dan masing-masing memiliki fungsinya tersendiri.

Partikel dibagi menjadi 4 jenis :

Kakujoshi : Partikel kasusSetsuzokujoshi : Partikel konjungsiFukujoshi : Partikel adverbialShuujoshi : Partikel akhir kalimat

Beberapa contoh partikel seperti partikel Wa, No, Ni, Ka, O, De, To, Kara, Made, Mo, dll. Saya akan menjelaskan sedikit beberapa partikel Bahasa Jepang beserta fungsinya.

--

Pembahasan

1). Partikel は (Wa)

Partikel ini ditulis dengan Hiragana 'Ha', tapi dibaca 'Wa'. Fungsinya yaitu sebagai penanda topik kalimat. Dalam beberapa situasi, partikel ini juga dapat diartikan sebagai "adalah".

2). Partikel の (No)

Partikel ini memiliki banyak fungsi, tapi fungsi utama nya yaitu untuk menyatakan suatu kepemilikan. Misalkan pada kalimat 'A no B', maka dalam Bahasa Indonesia artinya seperti "B (milik) A".

3). Partikel に (Ni)

Partikel ini juga memiliki banyak fungsi. Beberapa nya yaitu untuk menunjukkan waktu, letak/keberadaan sesuatu, tempat tujuan, dll.

4). Partikel か (Ka)

Partikel ini fungsi utamanya untuk membuat kalimat tanya. Biasa diletakkan di akhir kalimat, setelah kata 'desu'.

5). Partikel を (O)

Partikel ini ditulis dengan Hiragana 'Wo', tapi dibaca 'O'. Partikel ini berfungsi sebagai penanda objek. Biasa diletakkan di antara kata kerja dan kata benda atau objek.

6). Partikel で (De)

Partikel ini fungsinya beragam. Beberapa nya yaitu untuk menunjukkan tempat melakukan aktivitas, menyatakan keterangan cara/alat, menyatakan penyebab/alasan, sebagai penyambung kalimat, dll.

7). Partikel と (To)

Partikel ini memilki beragam fungsi, salah satunya sebagai kata hubung "dan" untuk kata yang kedudukannya setara. Misalkan pada kalimat 'A to B', maka dalam Bahasa Indonesia artinya seperti "A dan B".

8). Partikel から (Kara)

Partikel ini salah satu fungsinya untuk menunjukkan titik awal. Kalau dalam Bahasa Indonesia artinya seperti "dari". Partikel ini biasa dipasangkan dengan partikel Made.

9). Partikel まで (Made)

Partikel ini berfungsi untuk menunjukkan titik akhir. Kalau dalam Bahasa Indonesia artinya seperti "sampai". Partikel ini biasa dipasangkan dengan partikel Kara. Misalkan pada kalimat 'A kara B made', maka dalam Bahasa Indonesia artinya seperti "dari A sampai B".

10). Partikel も (Mo)

Partikel ini berfungsi untuk menyampaikan sesuatu yang bermakna setara. Kalau dalam Bahasa Indonesia artinya seperti "juga". Misalkan pada kalimat 'A mo B', maka dalam Bahasa Indonesia artinya seperti "A juga B".

32. apa saja partikel partikel yang ada di bahasa Jepang? Jelaskan​


Jawaban :

Partikel-partikel yang ada dalam Bahasa Jepang beberapanya seperti partikel Wa, partikel No, partikel De, partikel Ni, partikel E, partikel Wo/O, partikel To, partikel Ga, partikel Kara, partikel Made, dll.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Pengertian Partikel Bahasa Jepang

Partikel dalam Bahasa Jepang merupakan sebuah kata bantu kalimat, yang jumlahnya ada banyak dan memiliki fungsinya tersendiri. Kalau dalam Bahasa Jepang, partikel ini disebut dengan 助詞 (Joshi). Joshi dibagi menjadi 4 jenis kelompok, yakni sebagai berikut.

格助詞 (Kakujoshi) : P. Penanda kasus接続助詞 (Setsuzokujoshi) : P. Konjungsi副助詞 (Fukujoshi) : P. Keterangan終助詞 (Shuujoshi) : P. Akhir kalimat

Contoh-Contoh Partikel Bahasa Jepang

1. Partikel は (Wa)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi sebagai penanda subjek atau topik dari suatu kalimat. Kadang bisa diartikan sebagai "Adalah". Ditulis dengan huruf Ha, tapi dibaca Wa.

2. Partikel の (No)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menyatakan suatu kepemilikan. Karena itu partikel ini kadang bisa berarti "Milik", "Punya", "Nya", dll.

3. Partikel で (De)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan tempat melakukan suatu aktivitas, menunjukkan keterangan cara/alat, menyatakan sebab atau alasan, dan lainnya. Kadang bisa berarti "Di", "Dengan", "Karena", dll.

4. Partikel に (Ni)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan letak atau tempat keberadaan sesuatu, menunjukkan arah atau tujuan, menunjukkan waktu, dan lainnya. Biasa diartikan sebagai "Di", "Ke", "Pada", dll.

5. Partikel へ (E)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan arah dan orang yang dituju. Partikel ini memiliki arti "Ke" atau "Untuk". Ditulis dengan huruf He, tapi dibaca E.

6. Partikel を (Wo / O)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi sebagai penanda objek atau kata benda. Tidak memiliki arti. Partikel ini biasa diletakkan di tengah kata benda dan kata kerja. Ditulis dengan huruf Wo, tapi dibaca O (dalam Romaji bisa ditulis Wo maupun O).

7. Partikel と (To)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menyebutkan hal yang lebih dari satu, menunjukkan kata yang dimaksud sebagai lawan yang diajak, menunjukkan keterangan hal yang terjadi setelahnya, dll. Kadang bisa berarti "Dan", "Dengan", "Bersama", "Lalu", dll.

8. Partikel が (Ga)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menegaskan subjek dari suatu kalimat, menyambungkan kalimat yang berlawanan, dll. Biasanya tidak memiliki arti, namun saat menyambungkan kalimat yang berlawanan dapat berarti "Tapi".

9. Partikel から (Kara)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan titik awal dari sesuatu, dan menyatakan sebab atau alasan. Bisa berarti "Dari", "Karena", dll.

10. Partikel まで (Made)

Termasuk ke dalam kelompok Fukujoshi. Berfungsi untuk menunjukkan titik akhir dari sesuatu, dan menyatakan batasan yang lebih luas. Hampir selalu berarti "Sampai". Partikel ini kadang berpasangan dengan partikel Kara.

38. Jelaskan perbedaan partikel で dan に yang dalam bahasa Indonesia bisa diartikan sebagai "di"​


╭─────── ೄྀ࿐ ˊˎ╰─➤ JAWAB + PENJELASAN :

Kedua partikel ini dalam bahasa Indonesia bisa diartikan sebagai "di". Namun ada perbedaannya, yaitu :

・Partikel で (de) digunakan untuk menunjukkan "tempat aktivitas".

・Partikel に (ni) digunakan untuk menunjukkan "titik akhir".

Contoh :

公園でゴミを捨てます

= Kouen de gomi o sutemasu

= Membuang sampah di taman

*Taman di sini sebagai "tempat aktivitas" dari pelaku. Jadi, pelaku melakukan "membuang sampah" nya di taman.

=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

公園にゴミを捨てます

= Kouen ni gomi o sutemasu

= Membuang sampah di taman

*Taman di sini sebagai "titik akhir" dari sampahnya. Jadi, "sampah" nya dibuang di taman.

Keterangan :

公園 (kouen) = tamanゴミ (gomi) = sampah捨てます (sutemasu) = membuang

39. Quisss .....Sebutkan Partikel Partikel yang ada dalam bahasa jepang ....​


Pendahuluan

Dalam kalimat Bahasa Jepang terdapat sebuah kata bantu yang memiliki beragam fungsi, yang disebut dengan partikel atau 助詞 (Joshi). Partikel ini dapat dibagi menjadi empat jenis, yaitu :

格助詞 (Kakujoshi) : Partikel kasus接続助詞 (Setsuzokujoshi) : Partikel konjungsi副助詞 (Fukujoshi) : Partikel adverbial終助詞 (Shuujoshi) : Partikel akhir kalimat

Jumlah keseluruhan partikel ada sekitar 188 partikel, dan masing-masing memiliki fungsinya tersendiri. Yang akan saya bahas kali ini hanya beberapa partikel dasar saja.

Pembahasan

Partikel-partikel dalam Bahasa Jepang

1). Partikel は (Wa)

Partikel ini ditulis dengan Hiragana 'Ha', tapi dibaca 'Wa'. Fungsinya yaitu sebagai penanda topik kalimat. Dalam beberapa situasi, partikel ini juga dapat diartikan sebagai "adalah".

2). Partikel の (No)

Partikel ini memiliki banyak fungsi, tapi fungsi utama nya yaitu untuk menyatakan suatu kepemilikan. Misalkan pada kalimat 'A no B', maka dalam Bahasa Indonesia artinya seperti "B (milik) A".

3). Partikel に (Ni)

Partikel ini juga memiliki banyak fungsi. Beberapa nya yaitu untuk menunjukkan waktu, letak/keberadaan sesuatu, tempat tujuan, dll.

4). Partikel か (Ka)

Partikel ini fungsi utamanya untuk membuat kalimat tanya. Biasa diletakkan di akhir kalimat, setelah kata 'desu'.

5). Partikel を (O)

Partikel ini ditulis dengan Hiragana 'Wo', tapi dibaca 'O'. Partikel ini berfungsi sebagai penanda objek. Biasa diletakkan di antara kata kerja dan kata benda atau objek.

6). Partikel で (De)

Partikel ini fungsinya beragam. Beberapa nya yaitu untuk menunjukkan tempat melakukan aktivitas, menyatakan keterangan cara/alat, menyatakan penyebab/alasan, sebagai penyambung kalimat, dll.

7). Partikel と (To)

Partikel ini memilki beragam fungsi, salah satunya sebagai kata hubung "dan" untuk kata yang kedudukannya setara. Misalkan pada kalimat 'A to B', maka dalam Bahasa Indonesia artinya seperti "A dan B".

8). Partikel から (Kara)

Partikel ini salah satu fungsinya untuk menunjukkan titik awal. Kalau dalam Bahasa Indonesia artinya seperti "dari". Partikel ini biasa dipasangkan dengan partikel Made.

9). Partikel まで (Made)

Partikel ini berfungsi untuk menunjukkan titik akhir. Kalau dalam Bahasa Indonesia artinya seperti "sampai". Partikel ini biasa dipasangkan dengan partikel Kara. Misalkan pada kalimat 'A kara B made', maka dalam Bahasa Indonesia artinya seperti "dari A sampai B".

10). Partikel も (Mo)

Partikel ini berfungsi untuk menyampaikan sesuatu yang bermakna setara. Kalau dalam Bahasa Indonesia artinya seperti "juga". Misalkan pada kalimat 'A mo B', maka dalam Bahasa Indonesia artinya seperti "A juga B".
