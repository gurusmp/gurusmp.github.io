5. |2x+3|>|x+1| jawabannya dong MA kelas X


Diketahui:

|2x+3| > |x+1|

Ditanyakan:

x

Jawab:

|2x+3| > |x+1|

|2x+3|² > |x+1|²

4x² + 12x + 9 > x² + 2x + 1

3x² + 10x + 8 > 0

Cari batas

3x² + 10x + 8 = 0

3x² + 6x + 4x + 8 = 0

3x (x+2) + 4 (x+2) = 0

(3x+4)(x+2) = 0

x = -4/3 atau x = -2

Untuk menentukan intervalnya, subtitusikan bilangan antara -4/3 dan -2 ke persamaan.

Misal x = -5/3.

3(-5/3)² + 8(-5/3) + 8 ... 0

3(25/9) - (40/3) + 8 ... 0

75 - 120 + 72 ... 0

27 > 0

(Memenuhi)

Maka, -2 < x < -4/3.

22. Carilah kata yang termasuk Reguler verb dan Irregular verb masing-masing sebanyak 10 atau lebih Mapel : Bahasa InggrisKelas : X SMA/MAbantu kak cicih ​


Jawaban:

Regular Verb

- DIE (Meninggal) > DIED

- ADVANCE (Memajukan) > Advanced

- NEED (Membutuhkan) > Needed

- HAMPER (Menghambat)> Hampered

- LOW (Menguak)> Lowed

- CAUSE (Menyebabkan)> Caused

- REFUSE (Menolak)> Refused

- START (Memulai)> Started

- BETRAY (Mengkhianati) > Betrayed

- DEFEND (Melindungi) > Defended

- DROP (Menjatuhkan) > Dropped

Irregular Verb

- MET (Bertemu) < V1 = Meet

- LOST (Hilang) < V1 = Lose

- DO (Melakukan) > V2&3 = DID

- TOLD (Diberitahu) < V1 = Tell

- CAN (Bisakah) > V2 = Could

- BEGAN (Dimulai) < V1 = Begin

- SEE (Melihat) > V2 = Saw

- GOT (Dapat) < V1 = Get

- FELT (Merasakan) < V1 = FEEL

Penjelasan:

Jadi, simplenya, Regular Verb itu adalah kata kerja yang bisa diganti menjadi Simple Past Tense hanya dengan menambahkan -ed atau -d saja.

Sementara Irregular Verb, kata kerja ini bisa berubah bentuknya. seperti See yang berubah menjadi Saw pas di Simple Past Tense. Juga seperti Drink menjadi Drunk

38 = n1.37 + 30.39 / n1+30
38(n1+30) = 37n1 + 1170
38n1 + 1140 = 37n1 + 1170
38n1 - 37n1 = 1170 - 1140
n1 = 30
