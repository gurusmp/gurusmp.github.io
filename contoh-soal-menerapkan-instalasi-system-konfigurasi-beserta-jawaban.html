a. Wujudnya gas

b. Energi ionisasinya

c. Afinitas elektronnya besar

d. Merupakan gas monoatomic

Pembahasan:

Titik didih dan titik leleh gas mulia makin tinggi dengan makin besarnya nomor atom. Titik didihnya beberapa derajat dibawah titik lelehnya. Titik didih dan titik leleh gas mulia sangat rendah hal tersebut menunjukkan bahwa gaya tarik menarik antar atom (ikatan van der waals) sangat lemah.

Helium merupakan zat yang titik didihnya paling rendah dibandingkan dengan semula zat di alam semesta. Titik leleh Helium (-2720) mendekati suhu mutlak.

Gas mulia memiliki titik didih dan titik leleh yang sangat rendah oleh karena itu di alam gas mulia berwujud gas. Kestabilannya yang tinggi juga menyebabkan gaya tarik-menarik antar atom-atomnya lemah sekali. Karenanya unsure-unsur gas mulia dalam keadaan biasa (suhu normal) berfasa gas.

Jadi, lemahnya gaya tarik-menarik menyebabkan titik didih dan titik lelehnya sangat rendah.

Berdasarkan asas Aufbau, pengisian elektron pada orbital dimulai dari subkulit yang memiliki tingkat energi lebih rendah ke tingkat energi yang lebih tinggi.

Contoh :

12Mg = 1s2 2s2 2p6 3s2Berdasarkan aturan Hund, jika elektron tersebar dalam orbital dengan tingkat energi yang sama, elektron tidak akan berpasangan sebelum orbital tersebut terisi penuh.

Contoh :

Manakah pernyataan di atas yang memenuhi aturan Hund?

Yang memenuhi aturan Hund adalah penulisan pada nomor (ii), karena elektron harus mengisi seluruh orbital terlebih dahulu sebelum berpasangan.

40. macam-macam aplikasi system operasi beserta contohnya!


Pengertian Operating System (Sistem Operasi)
Secara mudah saja kita sebagai pengguna menggunakan komputer melalui media aplikasi atau yang lebih kerap disebut dengan Software (perangkat lunak). Aplikasi-aplikasi dalam komputer ini bekerja menurut OS (Operating System) yang telah ditanamkan dalam komputer, jadi tanpa adanya OS maka komputer tidak dapat digunakan. Secara gambaran aplikasi bekerja menurut OS, dan OS bekerja menurut perangkat keras komputer. Kesimpulannya sistem operasi merupakan sebuah program yang mengatur dan mengelola perangkat-perangkat keras (hardware) dari komputer dan memberikan pelayanan (sumber daya) terhadap aplikasi-aplikasi perangkat lunak. Sekarang anda bisa menyimpulkan bahwa Monitor, keyboard, speaker dan semua perangkat keras yang terdapat dalam komputer sepenuhnya dikendalikan oleh OS. OS juga memberikan pelayanan terhadap aplikasi-aplikasi yang ingin menggunakan Hardware dari komputer namun terbatas berdasarkan OS yang digunakan, misalnya aplikasi pemutar musik yang mengendalikan speaker.

Apa saja macam sistem operasi untuk komputer yang sudah ada saat ini?
Sistem operasi komputer saat ini sudah ada banyak, tak hanya komputer bahkan perangkat mobile seperti HP (Smartphone) kini juga telah menggunakan sistem operasi. Macam sistem operasi untuk HP yang populer contohnya: Android, Windows Phone, Symbian, dll. Macam-macam sistem operasi untuk komputer;

1. Windows
Microsoft Windows atau disebut dengan Windows saja merupakan salah satu sistem operasi yang ada saat ini. Windows menjadi sistem operasi yang sangat populer, bahkan bisa dikatakan hampir semua komputer pernah menggunakan OS ini. Dominasi Windows terhadap OS komputer menjadikannya semakin maju dan diperbaharui, bahkan untuk perangkat seluler telah menciptakan OS windows Phone. Kini Sistem Operasi mutakhir dari windows diberi nama Windows 8 setelah sebelumnya sukses dengan Windows 7. Beberapa versi dari windows diantaranya Windows 98, Windows 2000, Windows XP, Windows Vista, Windows 7, dan Windows 8.

2. Linux
Sistem Operasi Linux merupakan salah satu sistem operasi yang sukses. Meskipun tidak sesukes Microsoft Windows namun Berbagai perusahaan besar seperti Intel telah mengenal sistem operasi ini. Sistem operasi linux biasanya digunakan untuk server, Video game (Playstation 2, 3, 4), Ponsel, dan juga untuk komputer. Keunggulan sistem Operasi Linux dibanding windows yang paling mencolok adalah dari tingkat keamanannya. Secara gamblang akan banyak sekali orang yang bersemangat untuk mempelajari sistem operasi Windows karena hampir semua komputer menggunakan windows, namun akan sedikit yang mempelajari sistem operasi linux karena pengguna linux tidak sebanyak windows. Mungkin hal inilah yang menyebabkan sistem operasi linux dikatakan lebih aman dibanding sistem operasi windows, ibaratnya banyak yang pakai pasti banyak yang ingin mencuri sedangkan sedikit yang pakai sedikit pula yang ingin mencuri.

3. Mac OS
Anda mungkin sudah sering mendengar dengan yang namanya Windows dan Linux, namun bagaimana dengan Mac OS? Mac OS merupakan singkatan dari Macintosh Operating System, sistem operasi yang dibuat oleh aplle dan dibuat untuk komputer keluaran Macintosh.
