Diketahui data sebagai berikut:
51,5  45,5  52,5  63,5  50,5  58,5  61,5  88,5  70,5  77,5  81,5  63,5
n = 12
Data terurutnya:
45,5  50,5  51,5  52,5  58,5  61,5  63,5  63,5  70,5  77,5  81,5  88,5
a. Statistik lima serangkai
1) Data terkecil = 45,5
2) Data terbesar = 88,5
3) Median = Q2 = (61,5 + 63,5)/2 = 125/2 = 62,5
4) Kuartil pertama = Q1 = (51,5 + 52,5)/2 = 104/2 = 52
5) Kuartil ketiga = Q3 = (70,5 + 77,5)/2 = 148/2 = 74
b. Simpangan kuartil = 1/2 (Q3 – Q1) = ½ (74 – 52) = ½ (22) = 11
c. Letak desil ke-i = i(n + 1)/10
Letak desil ke-3 = 3(12 + 1)/10 = 3(13)/10 = 39/10 = 3,9
D3 = x3 + 0,9(x4 – x3)
      = 51,5 + 0,9(52,5 – 51,5)
      = 51,5 + 0,9(1)
      = 51,5 + 0,9 = 52,4
Letak desil ke-6 = 6(12 + 1)/10 = 6(13)/10 = 78/10 = 7,8
D6 = x7 + 0,8(x8 – x7)
      = 63,5 + 0,8(63,5 – 63,5)
      = 63,5 + 0,8(0) = 63,5

Letak desil ke-8 = 8(12 + 1)/10 = 8(13)/10 = 104/10 = 10,4
D8 = x10 + 0,4(x11 – x10)
      = 77,5 + 0,4(81,5 – 77,5)
      = 77,5 + 0,4(4)
      = 77,5 + 1,6 = 79,1

3,4,4,4,4,5,5,6,6,6,6,7

Jadi mediannya = 5

Modus bilangan dengan frekuensi tertinggi pada sekumpulan data umum. Modus= angka  yang paling banyak muncul
