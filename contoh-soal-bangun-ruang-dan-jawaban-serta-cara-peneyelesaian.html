33. Tentukan himpunan peneyelesaian dari |2x - 1| = |x + 4| !!!​

|2x - 1|² = |x + 4|²

(2x - 1)² - (x + 4)² = 0

(2x - 1 + x + 4)(2x - 1 - x - 4) = 0

(3x + 3)(x - 5) = 0

x = -1 atau x = 5

(2x-1)^2 = (x+4)^2

(2x-1)^2 - (x+4)^2 = 0

(2x-1-x+4) (2x-1+x-4) = 0

(x+3) (3x-5) = 0

x+3 = 0 atau 3x-5 = 0

x = -3 3x = 5

x = 5/3
