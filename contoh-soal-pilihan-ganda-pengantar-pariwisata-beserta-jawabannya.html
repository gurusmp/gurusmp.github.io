Macam-Macam Bilangan Negatif

Bilangan negatif adalah macam-macam bilangan bernilai negatif. Contoh: N = { -3, -5, ¼, …. } Keterangn tambahan: -2/-3 = ⅔, berarti -2/-3 bukan termasuk bilangan negatif.

Macam-Macam Bilangan Positif

Bilangan positif adalah macam-macam bilangan yang bernilai positif selain nol. Contoh: P = { 2, 3, 4, ¼, …. }dinyatakan dalam bentuk a/b. Maka a dan b adalah bilangan bulat dan b ≠ 0. Bilangan a dapat dikatakan sebagai pembilang dan bilangan b sebagai penyebut.

Contoh: H = { ⅓, ⅔, ⅛, ⅝, ….. } Keterangan tambahan: 4/2 = 2, berarti 4/2 bukan termasuk pecahan.

F = G m1m2/ r2

Keterangan:

F = gaya gravitasi (N) (satuan gaya)

r1 = R (di permukaan bumi)
