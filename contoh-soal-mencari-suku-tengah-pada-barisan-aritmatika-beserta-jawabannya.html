23. diketahui barisan aritmatika 5,8,11,........,137 banyak suku barisan itu ganjil A cari suku tengahnya B suku keberapakah suku tengahnya C berapa banyka suku barisan itu


A Nilai suku tengah = 71

B Suku tengahnya  adalah suku ke-23

C Banyak suku barisan itu adalah 45.

Pembahasan

Barisan Aritmatika adalah  suatu barisan dengan selisih antara dua suku yang berurutan selalu tetap.

Rumus : [tex]\boxed{~Un~=~a+(n-1)b~}[/tex]

Deret Aritmatika adalah jumlah suku – suku barisan aritmatika

Rumus : [tex]\boxed{~Sn~=~\frac{n}{2}(a+Un)~}[/tex]

Atau [tex]\boxed{~Sn~=~\frac{n}{2}(2a+(n-1)b)~}[/tex]

Untuk Barisan Aritmatika bertingkat Rumus Umumnya adalah

[tex]\boxed{~Un~=~an^2+bn+c~}[/tex]

Penyelesaian Soal

Diketahui:

Barisan aritmatika 5, 8, 11, ... , 137

Ditanya:

A Cari suku tengahnya

B Suku keberapakah suku tengahnya

C Berapa banyka suku barisan itu

Jawab:

Ut = [tex]\frac{1}{2}[/tex] (a + Un)

    = [tex]\frac{1}{2}[/tex] (5 + 137)

    =  [tex]\frac{1}{2}[/tex] (142)

    = 71

Ut = a + (t - 1)b

71 = 5 + (t - 1)3

71 = 5 + 3t - 3

71 = 3t + 2

71 - 2 = 3t

3t = 69

t = 69 : 3

t = 23

Un = a + (n - 1) b

137 = 5 + (n - 1)3

137 = 5 + 3n - 3

137 = 3n + 2

137 - 2 = 3n

3n = 135

n = 45

A Nilai suku tengah = 71

B Suku tengahnya  adalah suku ke-23

C Banyak suku barisan itu adalah 45.

Penjelasan dengan langkah-langkah:

Pembahasan :

Barisan Bilangan

Barisan bilangan adalah susunan bilangan yang diurutkan dengan aturan tertentu atau pola tertentu, bentuk barisan bilangan adalah a1, a2, a3, …, an.

Keterangan :

Setiap bilangan yang berada pada barisan bilangan disebut dengan suku (U)Suku pertama dilambangkan dengan a atau U₁Suku kedua dilambangkan dengan U₂Suku ketiga dilambangkan dengan U₃ sampai dengan suku ke-n (Uₙ)

Barisan Aritmatika

Barisan Aritmatika adalah barisan bilangan yang selisih antara tiap suku berurutan selalu tetap. Selisih antara dua suku yang berurutan pada barisan aritmatika sering disebut dengan "beda" dan dilambangkan dengan b. Secara umum, beda barisan aritmatika dirumuskan sebagai berikut :

b = Uₙ - Uₙ₋₁

Setiap suku barisan aritmatika (kecuali suku pertama) merupakan hasil penjumlahan suku sebelumnya dengan beda. Sebagai contoh, suku kedua merupakan hasil penjumlahan suku pertama dengan beda, suku ketiga merupakan hasil penjumlahan suku kedua dengan beda, dan seterusnya.

Jadi rumus untuk mencari suku ke-n barisan aritmatika adalah :

Uₙ = a + (n-1)b

Keterangan : Uₙ = suku ke-n

                       a = suku pertama

                       b = beda

                       n = banyaknya suku

Deret Aritmatika

Deret aritmatika adalah penjumlahan suku-suku dari suatu barisan aritmatika. Penjumlahan dari suku-suku petama sampai suku ke-n barisan aritmatika dapat dihitung.

Rumus :

Sₙ = n/2(2a + (n-1) b)

Keterangan : Sₙ = Jumlah suku n pertama

                       a = suku pertama

                       b = beda

                       n = banyaknya suku

Penyelesaian Soal :

diketahui : suku ke-4 (U₄) = 15

                  Suku tengah (Ut) =24

                  Suku terakhir (Un) = 49

ditanya : suku pertama (a), beda (b), dan banyaknya suku pada barisan ?

jawab :

untuk mencari suku pertama (a) dan beda (b) dengan cara mencari dari rumus suku tengah (Ut) :

Ut = 1/2 (a + Uₙ)                       substitusikan nilai a untuk mencari b :

24 = 1/2(a + 49)                        U₄ = a + 3b

24 × 2 = a + 49                         15 = (-1) + 3b

48 = a + 49                                15 + 1 = 3b

a = 48 - 49                                  16 = 3b

a = -1                                             b = 16/3

kemudian hitung banyaknya suku dalam barisan, dengan cara :

Uₙ = a + (n-1)b

49 = (-1) + (n-1) (16/3)

49 + 1 = (n-1) (16/3)

50 = (n-1) (16/3)

n - 1 = 50 × 3/16

n - 1 = 150/16

n = 150/16 + 1

n = 166/16

n = 83/8

n = 10,375 ≈ 10

∴ Kesimpulan nilai suku pertama (a) dan beda (b) adalah -1 dan 16/3, sedangkan banyaknya suku pada barisan tersebut adalah 10.

29. Saku tengah dan barisan aritmatika 80, 77, 74, ...... 20 adalah Dan tentukan juga n dari suku tengah tersebut..​


Barisan aritmatika di atas memiliki suku-suku sebagai berikut: 80, 77, 74, ... 20.

Saku tengahnya adalah suku ke-n/2, dengan n adalah banyaknya suku pada barisan tersebut.

N dapat ditemukan dengan rumus: n = (bawah range - atas range + jarak)/jarak.

Dengan bawah range adalah 20, atas range adalah 80, dan jarak adalah -3, maka n = (80-20)/(-3) + 1 = 26.

Sebagai suku ke-n/2, suku tengah barisan aritmatika tersebut adalah suku ke 13, dengan nilai 74.

Jadi, saku tengah dari barisan aritmatika tersebut adalah 74 dan n adalah 26.
