19. MEMBUAT BAGIAN² MICROSOFT WORD BESERTA FUNGSINYA​


Jawaban:

Microsoft Word adalah salah satu program pengolah kata yang paling populer dan sering digunakan di seluruh dunia. Berikut ini adalah beberapa bagian utama dalam Microsoft Word beserta fungsinya:

Menu Ribbon: Menu Ribbon terletak di bagian atas jendela Microsoft Word dan terdiri dari beberapa tab, seperti Home, Insert, Page Layout, References, Review, dan lain-lain. Setiap tab berisi grup perintah yang terkait dengan fungsi tertentu, seperti formatting teks, menyisipkan gambar, mengatur tata letak halaman, dan sebagainya.

Toolbar: Toolbar terletak di bawah Menu Ribbon dan berisi ikon-ikon perintah yang sering digunakan, seperti perintah untuk menyimpan dokumen, mencetak, mengubah format teks, dan lain-lain. Toolbar dapat disesuaikan sesuai dengan preferensi pengguna.

Area Kerja Dokumen: Ini adalah area utama di mana pengguna dapat membuat, mengedit, dan memformat dokumen. Pengguna dapat menulis teks, menyisipkan gambar, tabel, grafik, dan mengatur tata letak halaman di area kerja dokumen.

Ruler: Ruler terletak di atas dan samping area kerja dokumen. Ruler memungkinkan pengguna untuk mengatur tata letak dan format dokumen dengan mudah, seperti mengatur margin, tabulasi, dan paragraf.

Pane Navigasi: Pane Navigasi terletak di sisi kiri atau kanan jendela Microsoft Word. Pane Navigasi memungkinkan pengguna untuk dengan mudah menavigasi dan mengorganisir dokumen, seperti melompat ke halaman tertentu, melihat daftar isi, mencari teks, dan lain-lain.

Spell Check: Spell Check adalah fitur yang secara otomatis memeriksa ejaan kata dalam dokumen dan memberikan saran perbaikan jika ada kata yang salah dieja. Fitur ini membantu pengguna untuk menghindari kesalahan pengejaan yang umum.

AutoCorrect: AutoCorrect adalah fitur yang secara otomatis memperbaiki kesalahan pengetikan umum saat pengguna mengetik. Misalnya, jika pengguna mengetik "tehunng" secara otomatis akan dikoreksi menjadi "tahun".

Styles and Formatting: Styles and Formatting memungkinkan pengguna untuk menerapkan format teks yang konsisten di seluruh dokumen, seperti mengubah jenis font, ukuran, warna, dan efek teks dengan cepat.

Mail Merge: Mail Merge adalah fitur yang memungkinkan pengguna untuk membuat surat atau dokumen yang dikustomisasi dengan menggunakan data dari sumber eksternal, seperti daftar kontak atau basis data.

Track Changes: Track Changes memungkinkan pengguna untuk melacak perubahan yang dibuat dalam dokumen oleh pengguna lain. Fitur ini sangat berguna saat melakukan kolaborasi atau revisi dokumen.

Ini hanya beberapa bagian utama dalam Microsoft Word beserta fungsinya. Terdapat banyak fitur dan fungsi lainnya yang dapat digunakan dalam program ini untuk membuat dan mengelola dokumen dengan efisien.

24. cara membuat template pada microsoft word?​


Jawaban Dengan Langkah²

Dokumen perusahaan tidak melulu berisikan angka-angka dan tulisan. Ada kalanya gambar atau graphis pun perlu ada untuk menggambarkan besaran penjualan misalnya. Nah untuk memudahkan biasanya bagian administrasi sudah memiliki template sendiri bila ingin menyertakan gambar ke dalam laporan perusahaan.

Template Microsoft Word adalah dokumen Word yang menyertakan pengaturan khusus seperti teks boilerplate, macros, header dan footer, kamus khusus, toolbar, dan entri AutoText. Elemen-elemen yang sama akan selalu ada setiap kali Anda membuka template dan tidak dapat diubah. Dengan adanya template Anda juga sudah memotong durasi pengaturan file menjadi lebih cepat karena sudah ada template standar yang hanya perlu update tanggal maupun angka-angka nominal di dalamnya.

Merencanakan Template Word

Sebelum Anda membuat template Word Anda, buat daftar fitur yang ingin Anda sertakan di dalamnya seperti contoh di bawah ini:

Teks yang akan selalu disertakan dalam dokumen

Memformat kolom, margin, tab berhenti, catatan akhir, dan catatan kaki

Makro untuk mengotomatisasi tugas.

Bidang tanggal yang diperbarui secara otomatis setiap kali templat dibuka.

Alamat dan informasi kontak

Fields atau AutoText untuk informasi yang berubah seperti nomor halaman, judul dokumen, atau path file di header dan footer.

Teks placeholder dengan pemformatan tertentu seperti ukuran jenis huruf.

Setelah Anda menyusun semua elemen yang akan disertakan ke dalam template, buatlah prototipe dalam dokumen Word kosong dan lakukan penyesuaian bila dibutuhkan, dan jangan lupa simpan prototipenya.

Membuat template di Word 2019, 2016, 2013, DAN 2010

1. Buka File

2. Pilih Save As

3. Di kotak dialog Save As, di kotak teks File Name, ketik nama template

4. Kotak dialog Microsoft Word Save As menyimpan templat kustom

5. Jalur file berubah ke lokasi templat default. Templat di folder ini muncul di kotak dialog saat Anda membuat dokumen baru dari templat. Namun, Anda dapat memilih folder lain jika Anda mau

6. Pilih Save.

Membuat template di Word 2007

Anda bisa membuat template dengan cara berikut:

1. Pilih tombol Microsoft Office

2. Pilih Save As > Word Template

3. Di kotak dialog Save As, untuk menyimpan templat di folder templat default, di bawah Favorite Links pilih Template. Ini adalah folder yang disarankan karena templat di folder ini muncul di kotak dialog Template ketika Anda membuat dokumen baru berdasarkan templat. Namun, Anda juga dapat menavigasinya ke folder lain jika Anda mau.

4. Ketikkan nama deskriptif untuk template, lalu Save.

Membuat template di Word 2003

1. Pilih File > Save As

2. Ketikkan nama yang dapat dikenali untuk templat lalu Save as type dan pilih Document Template

3. Pilih Save.
