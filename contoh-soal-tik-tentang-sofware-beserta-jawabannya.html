12. buatlah soal tik( kelas 7) semester 2 sebanyak 10 berserta jawabannya ?​


MATERI: MICROSOFT OFFICE (WORD)

1. Apa fungsi CTRL + C?

Fungsi CTRL + C (copy) adalah untuk menyalin huruf, file, dsb.

2. Apa fungsi CTRL + V?

Fungsi CTRL + V (paste) adalah untuk menempelkan hal yang sudah dicopy sebelumnya.

3. Apa fungsi CTRL + B?

Fungsi CTRL + B (bold) adalah untuk menebalkan huruf / angka.Contoh: ABC

4. Apa fungsi CTRL + I?

Fungsi CTRL + I (italic) adalah untuk memiringkan huruf / angka.Contoh: ABC

5. Apa fungsi CTRL + U?

Fungsi CTRL + U (underline) adalah untuk memberikan garis bawah dibawah huruf / angka.Contoh: ABC

6. Apa fungsi CTRL + P?

Fungsi CTRL + P menampilkan form untuk print dokumen word. Di CTRL + P juga ada opsi untuk mengatur ukuran kertas yang akan digunakan.

7. Apa fungsi CTRL + N?

Fungsi CTRL + N adalah untuk membuat dokumen baru.

8. Apa fungsi subscript?

Subscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di bawah.Contoh: [tex]X_{2}[/tex]  (angka "2" berada di bawah)

9. Apa fungsi superscript?

Superscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di atas.Contoh: [tex]X^{2}[/tex]  (angka "2" berada di atas)

10. Sebutkan guna Microsoft Word!

24. contoh interaksi antar sofware- hardware -brainware​


Penyelesaian

Contoh interaksi antara software-hardware-brainware adalah ketika seorang yang sedang menulis suatu dokumen dengan mengggunakan software Microsoft Word dengan bantuan keyboar. Soal tersebut merupakan soalt tentang perangkat lunak pada komputer.

Pembahasan

Soal tersebut merupakan soal tentang perangkat lunak komputer. Perangkat lunak komputer merupakan bagian dari komputer yang di dalamnya berisikan perintah-perintah yang ingin dilakukan oleh pengguna. Ketika menggunakan komputer, tentunya kita tidak dapat lepas dari interaksi antara 3 kompunen penting yaitu software-hardware-brainware. Berikut akan dijelaskan masing-masing pengertian dari software-hardware-brainware.

Software

Software merupakan kumpulan instruksi komputer yang digunakan untuk menjalankan tugas-tugas tertentu pada perangkat keras komputer. Hal ini mencakup sistem operasi, aplikasi, dan program-program lain.Contoh dari software adalah Microsoft Word digunakan untuk mengolah kata, Microsoft Excel digunakan untuk mengolah data dalam bentuk tabel, Microsoft Powerpoint digunakan untuk membuat presentasi.

Hardware

Hardware merupakan komponen fisik dari komputer dan perangkat keras lainnya yang digunakan untuk menjalankan software. Contoh dari hardware adalah keyboar, monitor, CPU, mouse.

Brainware

Brainware merupakan manusia yang berinteraksi dengan perangkat keras dan software. Contoh dari brainware adalah pengetahuan, keterampilan, dan kemampuan individu dalam menggunakan teknologi komputer.

34. contoh sofware yaitu​


Jawaban:

Software dikelompokkan dalam dua klasifikasi, yaitu berdasarkan jenis dan distribusi. Ulasannya sebagai berikut:

1. Software Berdasarkan Jenis

Ada beberapa jenis software atau perangkat lunak di dalam sebuah sistem operasi komputer, di antaranya:

Application Software (perangkat lunak aplikasi). Software ini punya fungsi tertentu, misalnya untuk presentasi dan pengolahan data. Contoh application software: Microsoft Office Word, Microsoft Office Excel, MYOB, OpenOffice.org.

Operating System (sistem operasi). Fungsi software ini biasanya digunakan untuk mengkoordinasikan tiap komponen dan fungsi komputer. Contoh: Windows, DOS, dan Linux.

Programming Language (bahasa pemrograman) adalah software yang berfungsi sebagai pemberi instruksi standar. Contoh: PHP, Java, dan Microsoft Visual Basic.

 

2. Software Berdasarkan Distribusi

Selain jenis-jenis software di atas, ada beberapa software lainnya yang diklasifikasikan berdasarkan distribusi, antara lain:

Freeware, yaitu software tanpa batas waktu tertentu. Biasanya, software jenis ini memiliki fitur kurang lengkap dan tidak maksimal.

Adware, yaitu software yang bisa didapatkan dan digunakan secara gratis namun dengan kompensasi adanya iklan yang muncul di komputer user.

Spyware, yaitu perangkat lunak yang dibuat khusus untuk memata-matai segala aktivitas pengguna komputer. Biasanya, software jenis ini banyak disalahgunakan, seperti untuk mencuri data dari komputer lain.

OpenSource, yaitu software yang kode sumbernya dapat dibuka, diubah-ubah, ditingkatkan, dan disebarluaskan. Biasanya software jesni ini dapat diperoleh secara gratis dan dapat dikembangkan oleh orang lain dengan lisensi GPL (General Public License).

Shareware, yaitu piranti lunak untuk keperluan tertentu yang dibagikan secara gratis, biasanya sebagai demonstrasi dengan fitur terbatas dan penggunaannya untuk waktu terbatas.

Malware, yaitu software yang dianggap sebagai perusak, sehingga bisa berbahaya jika software malware disalahgunakan.

Firmware, yaitu jenis software penyimpanan yang hanya dapat dibaca atau Memory Read Only. Software ini bersifat paten, sehingga tidak bisa dilakukan modifikasi atau pengembangan meskipun ada masalah dalam fungsinya.

35. Buat lah 15 soal tentang (TIK) perangkat lunak aplikasi, fitur dan bagian² tentang keyboard beserta jawaban nya ..​


Soal 1

Apa yang dimaksud dengan perangkat lunak aplikasi?

Jawaban:

Perangkat lunak aplikasi adalah perangkat lunak yang dirancang untuk menyelesaikan tugas tertentu, seperti mengetik, membuat presentasi, atau bermain game. Perangkat lunak aplikasi biasanya dibeli atau diunduh dari internet.

Soal 2

Apa saja jenis-jenis perangkat lunak aplikasi?

Jawaban:

Jenis-jenis perangkat lunak aplikasi dapat dikelompokkan berdasarkan fungsinya, seperti:

Perangkat lunak produktivitas, seperti Microsoft Office, Adobe Photoshop, dan Google Chrome.

Perangkat lunak hiburan, seperti game, pemutar musik, dan pemutar video.

Perangkat lunak pendidikan, seperti software pembelajaran bahasa, software pembelajaran matematika, dan software pembelajaran komputer.

Perangkat lunak bisnis, seperti software akuntansi, software manajemen proyek, dan software manajemen database.


Soal 3

Apa saja fitur-fitur yang umum terdapat pada perangkat lunak aplikasi?

Jawaban:

Fitur-fitur yang umum terdapat pada perangkat lunak aplikasi antara lain:

Fitur antarmuka pengguna, seperti menu, tombol, dan ikon.

Fitur pengeditan, seperti cut, copy, paste, dan undo.

Fitur format, seperti font, ukuran, dan warna.

Fitur kolaborasi, seperti berbagi file dan bekerja bersama.

Fitur keamanan, seperti proteksi kata sandi dan enkripsi data.


Soal 4

Apa saja bagian-bagian yang terdapat pada keyboard?

Jawaban:

Keyboard terdiri dari beberapa bagian, yaitu:

Bagian tombol, yaitu bagian yang digunakan untuk mengetik huruf, angka, dan simbol.

Bagian tombol fungsi, yaitu bagian yang digunakan untuk menjalankan perintah khusus.

Bagian tombol navigasi, yaitu bagian yang digunakan untuk menggerakkan kursor.

Bagian tombol kontrol, yaitu bagian yang digunakan untuk menjalankan perintah umum, seperti enter, delete, dan backspace.

Jawaban:

Tombol enter berfungsi untuk mengakhiri baris teks atau menjalankan perintah.

Soal 9

Soal 10

Soal 11

Soal 12
