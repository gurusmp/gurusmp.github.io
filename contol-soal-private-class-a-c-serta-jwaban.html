24. 12. Mengelompokkan class - class yang saling berelasi menggunakan perintah java yaitu.... a. Private b. Package c. Import d. Public e. interface


Penjelasan:

Perintah-perintah yang digunakan untuk mengelompokkan class-class yang saling berelasi dalam bahasa pemrograman Java adalah:

a. Private: Digunakan untuk mendeklarasikan anggota class (variabel, method) yang hanya dapat diakses dari dalam class itu sendiri.

b. Package: Digunakan untuk mengelompokkan class-class dalam satu paket (package) yang memiliki kesamaan fungsi atau tujuan. Class-class dalam satu package dapat saling berinteraksi dan menggunakan anggota-anggota yang memiliki akses default atau modifier package.

c. Import: Digunakan untuk mengimpor class-class atau package lain ke dalam sebuah class. Dengan mengimpor, class-class tersebut dapat digunakan dan diakses dalam class yang mengimpor.

d. Public: Digunakan untuk mendeklarasikan anggota class (variabel, method, class) yang dapat diakses dari class lain, baik dalam package yang sama maupun dari package lain.

e. Interface: Digunakan untuk mendefinisikan kontrak atau spesifikasi yang harus diikuti oleh class-class yang mengimplementasikannya. Interface memberikan kumpulan method-method yang harus ada dalam class-class tersebut.

Dengan menggunakan kombinasi perintah-perintah tersebut, kita dapat mengelompokkan class-class dalam hierarki yang jelas dan mengatur aksesibilitas anggota-anggota class sesuai dengan kebutuhan.

37. Pernyataan berikut terkait abstract class yang benar adalah a. Abstract class tidak bisa memiliki konstruktor b. Abstract class tidak bisa diinstansiasi c. Abstract class tidak bisa diturunkan d. Abstract class tidak bisa memiliki private method


Jawaban:

Abstract class tidak bisa diinstansiasi

Penjelasan:

Abstract class atau kelas abstrak adalah kelas yang terletak di posisi tertinggi dalam hierarki class. Class ini tidak dapat diinstansiasi karena masih bersifat abstrak. Class ini hanya berisi variabel umum dan deklarasi method tanpa detail penggunaannya (abstract method).
