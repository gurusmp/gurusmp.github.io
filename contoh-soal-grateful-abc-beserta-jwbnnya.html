Untuk pemfaktoran terdiri Dari ax², bx, c
Misalnya x² + 2x - 15
Maka a= 1. b= 2 dan c= -15
Cari mudahnya carilah angka jika dikali hasilnya -15(nilai c) dan jika ditambah hasilnya 2(nilai b).
.... x.... = -15
.... x.... = 2

Jika hanya berbentuk a² - 36
Itu maka nilai "b" nya adalah 0
Maka a² + 0 - 36
... x... = - 36
... x... = 0
Jadi
-6 x 6 = - 36
-6 + 6 = 0

Untuk pangkat jika dalam bentuk perkalian ➡ Ditambah
Jika dalam bentuk pembagian ➡Dikurang.
Pangka yang tidak bertuliskan seperti x saja itu berarti pangkatnya bernilai 1.

30. assalamu'alaikum Hyungbagi yg bisa ni jwbnnya dengan rumus ABC apaan ya?Trims:)​


Jawab: x = -1/2, 3

Penjelasan dengan langkah-langkah:

Rumus a b c adalah rumus pamungkas dalam persamaan kuadrat setelah cara pemfaktoran. Biasanya, ketika cara pemfaktoran sudah tidak bisa, biasanya memakai rumus a b c ini.

Rumusnya, jika ax² + bx + c = 0,

x = [ -b ± √b² - 4ac ] / 2a

Karena soalnya adalah 2x² - 5x - 3 = 0, maka

a = 2b = -5c = -3

Masukkan nilai a, b, dan c ke rumus abc,

2x² - 5x - 3 = 0

x = [ -b ± √b² - 4ac ] / 2a

x = [ 5 ± √5² - 4 x 2 x -3 ] / 2 x 2

x = [ 5 ± √25 + 24 ] / 4

x = [ 5 ± √49 ] / 4

x = [ 5 ± 7 ] / 4

Jadi, niali x adalah :

x = [ 5 + 7 ] / 4 = 12 / 4 = 3

x = [ 5 - 7 ] / 4 = -2/4 = -1/2

Jadi, akar dari persamaan 2x² - 5x - 3 = 0 adalah :

x = 3x = -1/2

Mari kita buktikan dengan pemfaktoran!

2x² - 5x - 3 = 0

(2x + 1)(x - 3) = 0

x = -1/2, 3

Jadi, terbukti kalau nilai x = -1/2 dan 3
